## Запуск

Запуск контейнера с сервисом:

```bash
cd ./llm_spam_detector
docker compose up
```

FastAPI оболочка доступна на порте 8000. Для запросов к LLM используется эндпоинт `/predict`

## Отправка запроса к сервису

Пример обращения к сервису с поомщью curl:

```bash
curl -X POST http://localhost:8000/predict -H "Content-Type: application/json" -d '{"sms_text": "ВЫ ВЫИГРАЛИ 10278307107 долларов!!! Срочно скажите код из СМС банка!"}'
```

Ответ LLM на запрос:
```
{"is_spam":true,"reason":"SMS содержит определенные признаки спама: реклама, мошенничество, фишинг, запрос личных данных, подозрительные ссылки и обещания лёгкого заработка. Использование кодов подтверждения и уведомлений от банков/сервисов может быть критическим."}
```

## Системный промпт

Шаблон промпта (zero-shot):
```
Классифицируй SMS-сообщение как 'спам'/'не спам'.
Признаки спама: реклама, мошенничество, фишинг, запрос личных данных, подозрительные ссылки.
Определи, является ли следующее SMS-сообщение спамом: \"{sms_text}\".
Ответь исключительно в виде JSON с двумя полями: 'is_spam' (булево, true если спам, false если не спам) и 'reason' (краткое объяснение решения)
```

В него вставляется текст SMS-сообщения, отправленный пользователем в запросе.

## Клиентский скрипт

Есть внешний Python-скрипт для отправки запросов к сервису. Использует модуль requests.

```bash
python3 client.py
```

Пример вывода (список входящих СМС задается в самом скрипте):

```
Входящее СМС: ВЫ ВЫИГРАЛИ 213070247 долларов!!! Перейдите по ссылке для получения приза!!!
Результат: is_spam=True, reason=Ссылка на приз отправлена с целью обмана или мошенничества. Это является спамом.

Входящее СМС: Привет, как дела? Что нового за сегодня?
Результат: is_spam=False, reason=Простое сообщение об истории.

Входящее СМС: Ваша карта заблокирована! Срочно подтвердите данные здесь: http://obmana.net/scam
Результат: is_spam=True, reason=Причины: реклама, мошенничество, фишинг. Основные причины: запрос личных данных, подозрительные ссылки.
```
